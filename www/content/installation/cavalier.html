Ergo-L
  • Meilleur que Bépo pour le français, meilleur que Dvorak pour l’anglais :
    charge des doigts plus équlibrée (auriculaires notamment),
    et beaucoup moins de digrammes de même doigt.
  • Meilleur que Qwerty pour coder :
    les symboles de programmation sont accessibles soit comme en Qwerty ANSI,
    soit via une couche AltGr spécifiquement optimisée pour cet usage
  • Préserve les raccourcis à une main et les chiffres en direct.
https://ergol.org/
Ergo-L

Une disposition de clavier « mobile first » :

  • toutes les lettres, mêmes accentuées, se font dans le pavé alpha de 3×10 touches ;
  • tous les symboles de programmation peuvent se faire dans le pavé alpha,
    via une couche AltGr optimisée pour l’enchaînement des symboles ;
  • compatible avec tous les claviers, même les plus compacts ;
  • limite les extensions de doigts, même sur les claviers standard.
https://ergol.org/
